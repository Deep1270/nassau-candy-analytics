# Nassau Candy Distributor Analytics Dashboard 🍬

## Project Overview
This project provides data-driven visibility into product profitability, division performance, 
and supply chain diagnostic tracking for Nassau Candy Distributor. 

## Live Application
🔗 [Click Here to View the Live Streamlit Dashboard]([https://your-app-url.streamlit.app](https://2snpxfqs4u3ngowzv7m8qy.streamlit.app/#cost-vs-sales-scatter-analysis))

## Project Architecture
- `data/`: Contains raw and cleaned order data.
- `notebooks/`: Comprehensive Exploratory Data Analysis (EDA) and Pareto calculations.
- `app.py`: Main execution file for the interactive Streamlit user interface.
- `reports/`: PDF versions of the Research Paper.

## Key Performance Indicators Trailed
- Gross Margin (%)
- Profit per Unit
- Pareto Revenue & Profit Contributions
